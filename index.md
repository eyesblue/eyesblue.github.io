# Eyes Blue Docs

記錄各個專案的建置流程、設定步驟與開發筆記。

[查看專案清單]({% link projects.md %}){: .btn .btn-primary }

## 這個網站怎麼用

- 每個專案各自有一頁，集中記錄建置、部署與設定步驟
- 共通內容依主題分類，方便整理環境設定與問題排查
- 新增專案時可直接套用既有模板，避免每次從零開始寫

此網站使用 [Just the Docs](https://just-the-docs.com/) 建立，並由 GitHub Pages 發布。
